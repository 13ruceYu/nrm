nrm
===

NPM registry manager. Fast switch between all NPM registries.


## Install

```
$ npm install -g nrm
```

## use

```
// show all registries
nrm list

// switch to cnpm 
nrm use cnpm

// go to home page
nrm home

```


## registries

* NPM
* cnpm
* strongloop
* european
* australia
* nodejitsu
